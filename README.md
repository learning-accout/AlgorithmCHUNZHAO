## 一、切题四件套
1. Clarification：弄清题目
2. Possible Solutions：找到所有的解法，比较时间/空间复杂度，找出最优解
3. Coding：写代码（多写）
4. Test Cases：列举几个测试用例
## 二、五毒神掌（每道题至少做5遍）
第一遍：
- 读题+思考（5～15min)
- 直接看解法（多解法，比较优劣）
- 背诵、默写好的解法
   
第二遍：
- 马上自己写
- 多种解法比较、体会、优化

第三遍：
- 一天后，重复做前一天做过的题目
- 根据对不同解法的熟练程度进行专项练习

第四遍：
- 一周后，反复回来练习相同的题目
- 针对不熟的题目进行专项练习

第五遍：
- 面试前一周恢复性训练
